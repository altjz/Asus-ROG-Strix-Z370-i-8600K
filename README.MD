# Asus Rog Z370-I UHD630 + 8600K EFI for macOS Mojave

##  硬件配置

| 硬件  | 型号                           | 备注        |
|-----|------------------------------|-----------|
| CPU | i5 8600K                      |           |
| GPU | UHD 630                       |           |
| 主板  | ASUS ROG STRIX Z370-I GAMING |           |
| 网卡  | 博通 BCM94352z                | 替换主板原来的网卡 |
| 内存  | 芝士奇 2133 8G x2              |           |
| 硬盘  | 东芝 TR200 480G             |  不能开启 TRIM         |
| 机箱  | 肖邦 迎广                     | 自带150w电源  |

## 支持特性

1. 兼容至 10.14.6
2. 支持 WIFI 和 蓝牙
3. 支持 DP + HDMI 双显示器
4. 支持 DP HDMI 声音输出
5. 休眠和睡眠都目前没有问题
6. 重启后 HDMI 不再需要重新插拔
7. CPU 变频支持

## 已知问题

1. 开机的时候，HDMI 会闪屏 （但不需要重新插拔）
